WordPress 博客一步一步部署教程（小白友好版）

我这次使用 WordPress 这个经典博客项目来一步步讲解，如何部署一个属于自己的博客。

虽然这些内容看起来很基础，但对不同领域的小白来说还是有一定难度的。所以我写了这篇超详细教程，每一步都配有设置步骤和截图说明，力求不遗漏任何细节，让小白也能一遍就学会怎么搭建和部署。

第一阶段：配置 Cloudflare CDN 与 SSL 证书

在开始操作服务器之前，我们先把域名解析和 HTTPS 加密处理好，避免后面出现重定向错误。

1. 域名解析

  1. 登录 Cloudflare 控制台，进入你的域名管理页面。
  2. 找到 DNS → 记录。
  3. 添加两条 A 记录：
    • @（主域名）
    • www
  4. 两条记录都指向你的 VPS IP 地址。
  5. 关键点：确保记录后面的黄色云朵图标是打开状态（启用 Cloudflare CDN 服务）。

2. 生成源服务器证书

  1. 在 Cloudflare 左侧菜单找到 SSL/TLS → 源服务器。
  2. 点击“创建证书”按钮，保持默认设置生成即可。
  3. 生成后会显示两段内容：
    • PEM（证书）
    • KEY（私钥）
  4. 把这两段内容复制并临时保存到记事本，后面马上要用。

第二阶段：宝塔环境配置

我这里为了节约步骤，直接使用宝塔的一键部署功能，里面有很多常见网站的一键式部署模板，非常适合小白。

1. 添加站点

  • 点击左侧导航栏的 网站。
  • 在“PHP 项目”分类下，点击添加站点。

2. 一键部署 WordPress

  1. 切换到 一键部署 选项卡。
  2. 选择 WordPress 6.9（版本可能不是最新的，后续可自行升级）。
  3. 在“域名”输入框中填入你的域名。
  4. 其他选项保持默认，点击部署。

部署完成后会弹出数据库用户名和密码，请截图保存。
如果不小心关掉了也没关系，后续可以在“数据库”菜单里找到对应的数据库，获取账号和密码。

3. SSL 证书配置

  1. 点击刚刚创建好的网站，进入设置。
  2. 找到 SSL 菜单。
  3. 把刚才在 Cloudflare 生成的证书（PEM）和私钥（KEY）分别复制到对应位置（注意不要搞反）。
  4. 开启强制 SSL，让网站全局走 HTTPS 协议。

4. 设置伪静态规则

  1. 找到 伪静态 菜单。
  2. 在下拉列表中选择 WordPress。
  3. 点击保存。

第三阶段：安装 WordPress

后面就是点点点就能完成的简单操作了。

1. 选择语言

访问你的域名，会自动跳转到 WordPress 安装向导。
选择 简体中文 → 点击继续。

2. 欢迎页面

直接点击 继续。

3. 填写数据库信息

  • 数据库名
  • 用户名
  • 密码
    其他保持默认即可。

补充：如果没保存数据库账号密码也没关系。
进入宝塔左侧 数据库 菜单，找到和你域名一致的数据库，复制账号（表名和账号名一致）和密码。

4. 运行安装

数据库信息填写正确后，会看到“不错。您完成了安装过程…”的提示，点击 运行安装程序。

5. 设置网站基本信息

填写以下内容：

  • 站点标题
  • 管理员用户名
  • 登录密码（自动生成的密码请务必保存好）
  • 邮箱

6. 登录后台

使用刚才设置的账号（或邮箱+密码）登录。
登录成功后即可进入 WordPress 后台，开始设置自己的个人内容啦！

第四阶段：配置自动备份（必须做！）

网站搭建好后，数据备份是重中之重。我们用宝塔的计划任务功能，每天自动备份网站文件和数据库。

1. 添加网站文件备份

  1. 找到 计划任务 菜单。
  2. 选择 备份网站。
  3. 默认备份所有网站，保留最新 3 份（节省硬盘空间）。

2. 添加数据库备份

同样方式，选择 备份数据库，设置默认参数后确认。

到这里，你的 WordPress 网站就已经部署完毕！

  • 已配置 Cloudflare CDN 加速
  • 全站 HTTPS 加密
  • 完善的伪静态规则
  • 每天自动备份的安全保障

后续安装插件、主题优化等操作，就看你有没有时间慢慢折腾啦～
